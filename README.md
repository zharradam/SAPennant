# SA Pennant Player Search

A web application for searching SA golf pennant player match history, 
leaderboards, and handicap tracking across 2021–2026.

## Tech Stack
- **Frontend:** Angular 19, hosted on GitHub Pages
- **API:** ASP.NET Core .NET 10, hosted on Azure App Service
- **Database:** Azure SQL
- **Data Source:** Golf SA / Golfbox
